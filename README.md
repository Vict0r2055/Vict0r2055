### üß† MSc Research: Morpheme-Aware Tokenization & isiZulu LLM

Building a morpheme-aware tokenization pipeline and adapting Large Language Models for isiZulu, a low-resource, morphologically rich Nguni language.

- Curating a balanced isiZulu corpus from literature, media, government and educational sources  
- Text preprocessing: normalization, deduplication, filtering and cleaning  
- Designing a morphology-aware pipeline using **canonical segmentations**, noun class prefixes, verb extensions and agglutinative structure  
- Integrating **morpheme-aware BPE / constrained subword tokenization** to respect morpheme boundaries  
- Training / adapting LLaMA-style models for isiZulu using custom tokenizers  
- Evaluating tokenization and language modelling using:
